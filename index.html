Setup

A straightforward path from install to usable org chart

Instead of building an internal org chart solution from scratch, teams can install the app, configure the hierarchy, load data, and start exploring.

1

Install

Deploy the app in Salesforce and assign the Org Chart Explorer admin access needed for setup.

2

Configure

Choose your hierarchy object, parent relationship, and display fields for the org chart experience.

3

Load data

Use your own hierarchy data or import data into the packaged position object using the included tools.

4

Explore

Search people or positions, validate hierarchy structure, and launch a live org chart experience for users.

Org Chart Explorer is built for Salesforce teams that want a credible, packaged org chart experience without taking on the cost and maintenance burden of building one themselves.
